# How to name branches for jira integration
- Take the label of your ticket ticket_label => (e.g. "feat", "patch")
- Also grab the ticket name: ticket_name => (e.g. "DEV-8", "DEV-20")
- Name it as such: ```{ticket_label}/{ticket_name}/{anything}```

This is just a simple guide on how to make tickets appear in Jira so we can review them easily.
Naming conventions and source control after the initial ticket labels and name should be discussed amongst devs.
